Вкладка "Конструирование"

Вкладка конструирование предназначена для создания алгоритма для его дальнейшего запуска в режиме моделирования. Алгоритм по умолчанию состоит из 3 состояний и 3 символов (единицы, разделителя и затирателя).

Сверху располагается панель инструментов, на которой расположены функциональные кнопки: сохранение алгоритма(1), загрузка алгоритма из файла(2), задание алфавита(3), переход к моделированию(4) - и выпадающий список для выбора состояний(5).

Слева располагается панель инструментов для работы создания алгоритма: удалить состояние(1), добавить состояние слева(2) и справа(3), очистить ячейку таблицы(4)

Вклакда "Моделирование"

Вкладка "Моделирование" прездназначена для моделирования работы Машины Тьюринга.

Сверху располагается панель инструментов, на которой расположены функциональные кнопки: загрузить алгоритм (1), настроить параметры визуализации(2) - и выпадающий список для выбора одного из стандартных алгоритмов(3): суммирования и вычитания чисел. Также сверху располагается переключатель для сохранения трассы в файл и выпадающие списки для выбора операндов.

Слева расположена панель инструментов для управления процессом моделирования. По завершении работы алгоритма система выведет результат работы алгоритма. Снизу расположена лента Машины Тьюринга.
